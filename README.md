# 🧠 Artificial Neural Networks Practice

## 1. Environment Setup ⚙️

- To create and activate the virtual environment, run the following script from the root folder:
  ```
  ./setup_venv.ps1
  ```
  This will set up all necessary dependencies for the project.

---

## 2. Running Activity Tests 🧪

- For each activity, run the corresponding PowerShell script (`.ps1`) to execute and validate your solutions.  
  Example:
  ```
  ./01_red_neuronal_artificial.ps1
  ./02_red_pequena.ps1
  ```
  Make sure to run these scripts from the root folder.

---

## 3. Submission Format 📋

- You must complete all `TODO` instructions described in the `README.md` of each activity, located in the `exercises` folder corresponding to the assignment.
- Ensure that all tasks and requirements are addressed before submitting.

---

## 4. Submitting Your Assignment 🚀

- To submit your work, make a commit with the following format:
  ```
  feature/<your_name>
  ```
  Replace `<your_name>` with your actual name.  
- Push your changes to your forked repository.
