## 1. Theoretical Framework 📚

**Artificial Neural Networks (ANNs)** are computational models inspired by the human brain. They consist of nodes called *neurons*, organized in layers (input, hidden, and output), which process information through weighted connections (synaptic weights). Each neuron receives input signals, applies weights, adds a *bias*, and uses an activation function (such as the sigmoid) to produce an output. ANNs are widely used in classification, regression, pattern recognition, and machine learning tasks due to their ability to model complex relationships in data.
