## Introduccion

Las redes neuronales artificiales (RNA) son modelos computacionales inspirados en el funcionamiento del cerebro humano. Están compuestas por unidades llamadas *neuronas artificiales*, organizadas en capas, que procesan información mediante conexiones ponderadas y funciones de activación.

Este proyecto contiene ejercicios, diagramas y estructuras de código diseñadas para introducir los conceptos fundamentales de las RNA, incluyendo:

- Perceptrones simples y multicapa
- Propagación hacia adelante (*forward pass*)
- Funciones de activación como la sigmoide
- Diagramas didácticos con conexiones cruzadas
- Evaluación automatizada de estructuras neuronales

Cada ejercicio está acompañado de comentarios explicativos, estructuras visuales en ASCII, y rutinas reproducibles para facilitar el aprendizaje técnico y conceptual.
